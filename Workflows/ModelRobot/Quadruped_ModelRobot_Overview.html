Quadruped Robot Modeling with Simscape™

This repository contains exercises to help users learn concepts required to model a quadruped robot in Simscape.

Exercises

  1. Bouncing Ball: Model, Solution
  2. Pendulum: Model, Solution
  3. Double Pendulum: Model, Solution
  4. Hopping Leg: Model, Solution
  5. Jumping 2 Legs: Model, Solution
  6. Walk Line: Model, Solution
  7. Shoulder UJoint: Model, Solution
